# Vélo'v Lyon Analytics Lab

This repository contains the full work for the Vélo'v Dynamic Web Lab 2026, organized into three distinct parts.

## Structure

### Part 1: MongoDB Queries
- **File**: `queries.js` (root) and `part1-mongodb/queries.js`
- **Focus**: Advanced MongoDB queries including aggregation pipelines, geospatial indexing, and data projection.
- **Run**: `mongosh 20263IFMongoLab queries.js`

### Part 2: Dynamic Web Page
- **Folder**: `part2-dynamic-page/`
- **Focus**: Client-side interactivity with DOM manipulation, theme switching, and SVG event handling.
- **Open**: `part2-dynamic-page/TP.html` in a browser.

### Part 3: Full Analytics Dashboard
- **Folder**: `part3-web-app/`
- **Focus**: A professional-grade Fullstack application (Node.js/Express + MongoDB + Modern CSS).
- **Features**: Real-time stats, interactive map, demographic enrichment, and class-based architecture.
- **Run**: 
  ```bash
  cd part3-web-app
  npm install
  npm start
  ```

---

## Technical Highlights

- **Backend**: Express.js REST API with modular routing.
- **Frontend**: High-fidelity UI with Glassmorphism, CSS Grid, and responsive design.
- **Data**: MongoDB aggregation for complex analytics and demographic data enrichment.
